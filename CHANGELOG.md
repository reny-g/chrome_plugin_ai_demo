# 更新日志

本项目的版本更新记录维护在此文件中。

## [1.1.0] - 2026-06-09

### 新增

- 新增简历优化模式，并保留原有网页摘要模式。
- 支持上传、替换、查看和清除一份本地 Markdown 简历。
- 根据当前网页内容提取 JD 信息并生成两份完整 Markdown 简历：
  - 进阶简历：尽可能匹配岗位需求，对缺少事实依据的内容使用待补充提示。
  - 稳妥简历：仅基于原简历事实优化结构、重点和表达。
- 新增 JD 分析、差异说明、能力缺口和补充建议。
- 支持复制优化后的简历，并下载两份简历及分析报告。

### 优化

- 简历优化生成期间增加动态等待状态、已用时间和不确定进度动画，降低长请求被误判为卡死的概率。
- 收紧风险提醒的成立条件，区分生成可靠性风险与候选人信息缺口，并要求先进行语义等价判断。
- 优化生成提示词，要求尽量保留原简历的结构、标题层级、语气和排版风格。
- 将简历优化提示词拆分为独立版本化模块，便于后续迭代和问题诊断。
- 分层强化进阶版与稳妥版的事实边界，并规范无证据内容的待补充表达。
- 增加网页和简历提示词注入防护、精简 JSON 输出示例及输出前事实检查。
- 增加非招聘页面、内容截断、证据不足和事实风险提醒。
- 增加 AI JSON 输出结构及 JD 字段校验。
- 防止上传、替换或清除简历后，旧生成请求覆盖当前结果。

## [0.1.0]

### 新增

- 支持提取当前网页正文并生成 AI 摘要。
- 支持 OpenAI 兼容接口和 Chrome 内置 Summarizer API。
